# Vignettes
{: .fw-700 }

Step-by-step conceptual tutorials — from first principles to real-world case studies.
{: .fs-5 .text-grey-dk-100 }

---

The Vignettes section provides accessible, narrative-style explanations of MHASpread concepts. Each vignette is self-contained and builds on the previous one, guiding you from a conceptual introduction through to interpreting simulation outputs and reading published case studies.

---

## Learning Path

| Step | Vignette | Audience | Content |
|---|---|---|---|
| 1 | [**Introduction**](introduction) | All users | Non-technical overview of FMD and why simulation matters |
| 2 | [**Model Structure**](model_structure) | Technical | Detailed compartmental model walkthrough, SEIR algorithm |
| 3 | [**Transmission Processes**](transmission_processes) | Technical | Within-farm, spatial kernel, and movement-driven spread |
| 4 | [**Control Strategies Guide**](control_strategies) | Policy / Technical | Applying interventions, scenario profiles, decision trade-offs |
| 5 | [**Simulation Outputs**](simulation_outputs) | All users | Interpreting epidemic curves, spatial maps, uncertainty |
| 6 | [**Case Studies**](case_studies) | All users | Brazil, Bolivia, Chile real-world applications |

---

## Who Should Read What

**I am new to disease modeling:**  
Start with [Introduction](introduction), then [Simulation Outputs](simulation_outputs).

**I want to understand the mathematics:**  
Read [Model Structure](model_structure) and [Transmission Processes](transmission_processes), then cross-reference the [Model](../model) section.

**I am preparing a simulation run:**  
Check [Data Requirements](../data_requirements), then [Control Strategies Guide](control_strategies).

**I am evaluating policy options:**  
Start with [Control Strategies Guide](control_strategies) and [Case Studies](case_studies).
